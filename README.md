# my-personal-website

一个简洁的个人网站项目，适合展示个人介绍、技能和项目。

## 技术栈

- HTML
- CSS
- JavaScript

## 项目结构

```text
my-personal-website/
├── index.html
├── css/style.css
├── js/main.js
├── assets/
└── README.md
```

## 本地预览

1. 打开项目文件夹。
2. 双击 `index.html`。

## 部署到 GitHub Pages

1. 把项目上传到 GitHub 仓库。
2. 打开仓库 `Settings` -> `Pages`。
3. `Branch` 选择 `main`，`Folder` 选择 `/ (root)`。
4. 保存后等待几分钟即可访问。

## 可修改内容

- 在 `index.html` 修改文字内容。
- 在 `css/style.css` 修改页面样式。
- 在 `js/main.js` 修改交互逻辑。
